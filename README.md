# Neural Aurora Theme (Shopify)

A consciousness-serving Shopify theme for the **Museum of Impossible Things**.

## Features
- **Dark Mode First**: Optimized for `#1f2121` charcoal backgrounds.
- **Neural Gradients**: Teal and purple consciousness-aligned aesthetic.
- **Glassmorphism**: Exhibit cards with backdrop blur.
- **Alpine.js Interactions**: Typing animations and lightweight state.

## Installation
1. Zip this folder.
2. Upload to Shopify Admin > Online Store > Themes.
3. Customize via the "Neural Aurora Colors" settings panel.
